# User Manual

## Applicant

1. Login with `applicant1` or `applicant2`.
2. Open `My Profile` to update name, email and skills.
3. Open `Browse Jobs` to see all TA jobs.
4. Click `View Detail` to check the job description and match result.
5. Click `Apply Now` to submit an application.
6. Open `My Applications` to track the current status.

## MO

1. Login with `mo1`.
2. Open `My Jobs` to see jobs created by the MO.
3. Open `Create Job` to add a new TA position.
4. Click `View Applications` for a specific job.
5. Review applicant skills, match score and missing skills.
6. Change the status to `Pending`, `Accepted`, or `Rejected`.

## Admin

1. Login with `admin1`.
2. Open `Workload Overview`.
3. Review accepted job count and total hours for each applicant.
4. Check whether the system marks each applicant as `Normal` or `Overloaded`.
